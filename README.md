A bare metal operating system built from the scratch, targeting the 32 bit x86 architecture.

bastOs is a continuing project of a custom operating system, handling from booting procedures and hardware interrupts.

To compile and emulate bastOs, the required tools are needed (it will be handled.)

- 'i686-elf'
- 'i686-elf-gcc'
- GNU Binutils: 'i686-elf-as'
- QEMU
